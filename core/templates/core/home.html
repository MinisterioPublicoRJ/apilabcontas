Guia para cadastro de encio de dados ao MPRJ

Visando o intercâmbio de dados que propiciará o trabalho de inteligência integrado entre o Ministério Público e governo do Estado do Rio de Janeiro o parquet fluminense está disponibilizando para as secretarias de estado e seus parceiros uma ferramenta para recebimento automatizado de dados.

A ferramenta receberá atualização de dados em tempo real através do envio de um método POST HTTP para endereços web pré-definidos, um para cada modelo de informações.

Para ter acesso a ferramenta, as OSS deverão preencher o formulário linkado abaixo, informando o nome da instituição, a Unidade de saúde e o e-mail do responsável técnico pelo envio de dados. Após o preenchimento do formulário nossa equipe irá criar uma chave de autenticação para cada cadastro, e estas serão compartilhadas através do e-mail cadastrado acompanhadas de instruções técnicas sobre como deve ser utilizada a ferramenta para a atualização de dados em tempo real. O formulário está disponível neste link. Atenciosamente, Equipe MP em Mapas
