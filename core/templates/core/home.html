Visando o intercâmbio de dados que propiciará o trabalho de inteligência integrado entre o Ministério Público e o governo do Estado do Rio de Janeiro, o Parquet fluminense está disponibilizando para as secretarias de estado e seus parceiros uma ferramenta para recebimento automatizado de dados.

A ferramenta receberá atualização de dados em tempo real através do envio de um método POST HTTP para endereços web pré-definidos, um para cada modelo de informações.

Após o preenchimento do formulário, nossa equipe irá criar uma chave de autenticação para cada cadastro. As respectivas chaves serão enviadas para o e-mail cadastrado, acompanhadas de
instruções técnicas que detalharão como deve ser realizado o envio de arquivos para a atualização de dados em tempo real.
